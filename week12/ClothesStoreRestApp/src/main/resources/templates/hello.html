Hello world!

How to use this API:

T-Shirt

GET LIST T-Shirts:
1.GET LIST T-Shirts
2.GET /api/t_shirt
3.None
4.None(200 OK)

Response Body:
[
{
"id": 1,
"name": "Adidas men's t-shirt",
"color": "Black",
"price": 1999
},
{
"id": 2,
"name": "Puma men's t-shirt",
"color": "White",
"price": 1599
}
]
===================================================
GET SINGLE T-Shirt
1.GET SINGLE T-Shirt
2.GET /api/t_shirt/id
3.None
4.None(200 OK) or 404 NOT FOUND

Response Body 200:
{
"id": 1,
"name": "Adidas men's t-shirt",
"color": "Black",
"price": 1999
}

Response Body 404:
{
"message": "T-Shirt with this id wasn't found",
"timestamp": 1664285487577
}
====================================================
POST CREATE
1.POST CREATE
2.POST /api/t_shirt
3.Request Body:
{
"name": "Adidas men's t-shirt",
"color": "Black",
"price": 1999
}
4.200 OK OR 400 BAD REQUEST

Response status 200 OK!

Response Body 400:
{
"message": "color - Color should be between 2 and 30 characters;price - Price should be greater than 0;",
"timestamp": 1664285908732
}
========================================================
PUT UPDATE
1.PUT UPDATE
2.PUT /api/t_shirt/update/id
3.Request Body:
{
"name": "Adidas men's t-shirt",
"color": "Black",
"price": 1999
}
4.200 OK OR 400 BAD REQUEST

Response status 200 OK!

Response Body 400:
{
"message": "color - Color should be between 2 and 30 characters;price - Price should be greater than 0;",
"timestamp": 1664285908732
}
========================================================
DELETE T-Shirt:
1.DELETE T-Shirt
2.DELETE /api/t_shirt/delete/id
3.None
4.None(200 OK)
